Описание по установке пекеджа на телевизор Samsung Tizen

Имеется:

  • телевизор Samsung Tizen 4.0
  • Windows 10 (думаю, что на Mac всё то же самое верно)
  • пекедж package.wgt

Для установки пекеджа на телевизор необходимо проделать следующие шаги:

  1. Скачать и установить Tizen Studio with IDE
  2. Настроить пекеджи для работы с приложением на СмартТВ
  3. Создать сертификат
  4. Подключиться к телевизору через Tizen Studio
  5. Импортировать приложение в Tizen Studio
  6. Запуск приложения на телевизоре

Установка Tizen Studio

Необходимо скачать и установить Tizen Studio with IDE installer.

Я установил Tizen Studio 4.0 with IDE installer. Со временем Samsung выпускает новые версии, это в нашем случае не важно, могут быть отличия в интерфейсе

Настройка Package Manager

После успешной установки Tizen Studio необходимо открыть Package Manager (tools -> Package Manager)

и необходимо кликнуть по "Configuration" (на иконку с шестерёнкой)

Необходимо отключить автообновления

Далее, клик по "Extensions SDK", найти в списке "Samsung Tizen TV SDK" и отключить

Следующим шагом необходимо добавить extension нажав на иконку "плюс"

Заполняем соответствующие поля "name": "TV Extensions-3.0", "repository": "http://sdf.samsungcloudcdn.com/Public/smart_tv_sdk/releases/samsung_tizen_studio_tv_sdk/stv_ext_public/3.0" без кавычек

Нажимаем "Ok" и применяем новую конфигурацию

После обновления конфигурации кликаем по вкладке "Extension SDK" что бы установить все зависимости, которые включены в "TV Extensions-3.0"

Потребуется время, что бы все зависимости установились. После успешной установки всех зависимостей можно закрыть Package Manager и открыть Tizen Studio

Настройка сертификата

Для успешной установки пекеджа необходимо создать сертификат, это чистая условность, поля для заполнения в примере произвольные

Открываем Certificate Manager (в Tizen Studio tools -> Certificate Manager) и кликаем по иконке "плюс"

Последующие шаги просто иллюстрирую изображениями:

После того, как будет создан сертификат мы вернёмся к окну Certificate Manager, где уже автоматически будет выбран созданный сертификат

Этап создания сертификата завершён, закрываем Certificate Manager

Подключение к телевизору

Для создания подключения с телевизором необходимо знать IP-адреса телевизора и компьютера, на котором установлена Tizen Studio

В телевизоре необходимо перейти в раздел Apps и набрать циферный код "12345", после чего появится окно ввода IP-адреса компьютера, с которым необходимо создать подключение

Если это первое подключение в режиме Develop Mode, то необходима перезагрузка телевизора

С настройкой подключения на стороне телевизора закончено, переходим к Tizen Studio.

Для подключения телевизора необходимо в Tizen Studio перейти к tools -> Device manager и кликнуть по иконке Remote Device Manager

В появившимся окне кликаем по иконке "плюс" и вводим имя и IP-адрес телевизора:

Телевизор добавлен в список, осталось передвинуть ползунок "Connection" в положение "ON"

Подключение с телевизором создано. Окно Device Manager можно закрыть

Импорт приложения в Tizen Studio

Импортируем приложение в Tizen Studio: File -> Import как "Tizen Project"

На следующем шаге выбираем тип как "Archive file" и через "Browse" указываем путь до wgt-пекеджа

На следующем шаге устанавливаем "Profile" как "tv-samsung", "Version" в положение "3.0" и выставляем галочки:

После нажатия кнопки "Finish" возвращаемся назад к Tizen Studio, наблюдаем в левом столбце, во вкладке "Project Explorer" новый проект с импортированным wgt-пекеджем:

Запуск приложения на телевизоре

Что бы запустить приложение достаточно кликнуть правой кнопкой мыши по импортированному wgt-пекеджу, Run As -> Tizen Web Application

После чего приложение установится на телевизор и произойдёт запуск установленного приложения на телевизоре
